Welcome

Welcome to Designers Tools. This is a collection of various programs to help designers to make design process easy.

Screen Ruler

  • Click on Screen Ruler button
  • You can move screen ruler with Left Click + Drag
  • Use Right Click to see more options
    • Change measurement unit
    • Flip horizontal or vertical
    • Rotate at any degree
    • Change the ruler length
    • Change background color
    • Change foreground color
    • Opacity
    • Set ruler as always on top or normal window mode

Color Schemer

  • You can use the Color Picker button to pick color from any background on the whole screen
  • Or, experiment with various HSB values

Box Shadow Editor

  • Experiment with Box Shadow for CSS
  • And you can export to CSS as well

Snapshot Taker

  • Click on the Take Screenshot button. Program will automatically take screenshot
  • A new save file dialog will appear, here choose the file destination to save as PNG image

Linear Gradient Editor

  • You can edit linear gradients by adding and removing new color
  • There are sample gradients to choose and edit from
  • and you can export to
    • CSS: For Web
    • SVG: Also can be opened from Adobe Illustrator and Inkscape
    • Image files like PNG and JPG

Radial Gradient Editor

  • You can edit radial gradients by adding and removing new color
  • There are sample gradients to choose and edit from
  • and you can export to
    • CSS: For Web
    • SVG: Also can be opened from Adobe Illustrator and Inkscape
    • Image files like PNG and JPG

Background Image Editor

  • Edit pattern for Web as CSS
  • You can also export to Png image, SVG (Adobe Illustrator) and CSS (web) file format
